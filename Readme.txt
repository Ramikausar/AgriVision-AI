🌿 AI Plant Disease Detection System

🚀 How to Run the Project
Step 1 — Install Dependencies
pip install -r requirements.txt
Step 2 — Run the Application
python app.py in cmd, then wait until "Debugger is active" is shown(it takes 10 to 15 seconds)
Step 3 — Open the your Web-Browser and add this URL="http://127.0.0.1:5000"
(To stop the app, press Ctrl + C in the terminal)


📌 Overview

The AI Plant Disease Detection System is a machine learning-based application that detects plant health conditions using leaf images.

The system scans an image of a plant leaf and analyzes it using an AI model.
Based on the analysis, the system determines:

Whether the plant is healthy or diseased

If diseased, what disease the plant has

Recommended actions or treatments for the detected disease

This system helps farmers, researchers, and agriculture students quickly identify plant diseases and take appropriate action.

⚙️ How the System Works

User uploads a leaf image through the web interface.

The image is processed by the trained AI model.

The model classifies the leaf as:

Healthy

Diseased

If diseased, the system displays:

The disease name

Suggested solution or treatment
